# NK_RNA_ambient_v2 🧬

## Resumen / Summary
**ES**: Pipeline de bioinformática para el análisis de transcriptómica de células NK, enfocado en inmunosenescencia y corrección de RNA ambiental mediante `scCDC`. Este proyecto sigue el estándar de *Research Compendium*.

**EN**: Bioinformatics pipeline for NK cell transcriptomic analysis, focused on immunosenescence and ambient RNA correction using `scCDC`. This project follows the *Research Compendium* standard.

## Estructura / Structure
- `data/`: Datos crudos y procesados.
- `scripts/`: Flujo de trabajo numerado.
- `results/`: Tablas y figuras finales.
